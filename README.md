📒 Contact Book
A simple command-line Contact Book application built in Python.
Manage your contacts easily with a clean menu-driven interface —
all data is saved automatically so nothing is lost between sessions.

🚀 Features

➕ Add new contacts
✏️ Update existing contacts
👀 View all contacts
🔍 Search for a contact
🗑️ Delete a contact
💾 Auto-saves to a local JSON file


🛠️ Built With

Python 3
JSON (for data storage)


▶️ How to Run

Make sure Python 3 is installed on your system
Download or clone this repository
Run the script: python contact_book.py


📁 Project Structure
contact-book/
├── contact_book.py — Main application file
├── contacts.json — Auto-generated file that stores your contacts
└── README.md — Project documentation
